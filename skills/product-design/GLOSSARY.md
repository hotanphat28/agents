# Glossary of Terms
This document provides a single source of truth for terminology used within the `product-design` skill, ensuring context accuracy and preventing hallucinations.

## Styles & Principles
* **Brutalism**: A digital design style characterized by a raw, unpolished, and intentionally stark aesthetic, often featuring bold typography, clashing colors, and lack of refined spacing.
* **Gestalt Principles**: Psychological principles that explain how humans group similar elements, recognize patterns, and simplify complex images (e.g., proximity, similarity, continuity, closure).
* **UI/UX**: User Interface (the visual layout, elements, and interactive components) and User Experience (the overall feel, flow, and logic of a user's interaction with the product).
* **A11y (Accessibility)**: The practice of making web and mobile products usable by as many people as possible, including those with disabilities (vision, motor, auditory, cognitive).

## Systems & Components
* **Design Tokens**: The smallest, indivisible pieces of a design system (e.g., color hex codes, spacing units, font families) stored as variables to ensure consistency across platforms.
* **Component Library**: A centralized collection of reusable UI elements (buttons, inputs, modals) built from design tokens.
* **Design System**: A complete set of standards, documentation, and principles, along with a component library and design tokens, used to manage design at scale.

## Standards & Platforms
* **WCAG 2.2 AA**: Web Content Accessibility Guidelines version 2.2, conformance level AA. The industry standard baseline for legally compliant digital accessibility.
* **iOS HIG (Human Interface Guidelines)**: Apple's official design guidelines for creating apps that look and feel native to iOS, iPadOS, macOS, and watchOS.
* **Material 3**: Google's open-source design system, characterized by dynamic color (Material You), updated typography, and distinct component elevation models.

## Tools & Formats
* **Pencil.dev**: A specialized visual design and prototyping tool used within this AI agent ecosystem for creating and manipulating UI layouts natively via `.pen` files.
* **Figma**: The industry standard collaborative web-based interface design tool.
* **Sketch**: A vector graphics editor for macOS heavily used for UI/UX design.

## Game Design
* **HUD (Heads-Up Display)**: The method by which information is visually relayed to the player as part of a game's user interface (e.g., health bars, ammo, minimaps).
* **Color Script**: A visual roadmap of the colors and lighting in a game or film, used to map the emotional arc of the story to the visual aesthetic.
